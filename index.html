    O que é Nginx?

    O Nginx é um servidor web usado para hospedagem de sites e aplicações na internet, que pode ser utilizado como prozy reverso e balanceador de carga.
    Quando um site é acessado o navegador envia uma solicitação ao servidor que redireciona o trafego para outros servidores.
